Retrieves the rectangle for a character in the font tilemap.

Parameters
cThe character to retrieve the rectangle for.
Returns
An SDL_Rect specifying the position and size of the character in the tilemap.

This method calculates the source rectangle for a character c in the font tilemap based on its index and the tilemap layout. The tile size is assumed to be 16x16.
