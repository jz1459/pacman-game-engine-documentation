bool scene_manager::transitionToNextScene ( SceneTree sceneTree)

Transitions to the next scene (level).

Parameters
sceneTreeThe scene tree to manage game objects.

This method increments the scene index and sets the next scene as active. If there are no more scenes, it returns false to indicate the game should end. Otherwise, it returns true after successfully transitioning.
