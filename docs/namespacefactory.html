GameObject pacman,

Adds a movement component to a ghost game object.

Parameters
ghostThe ghost game object to add the movement component to.
pacmanThe Pac-Man game object that the ghost will target.
sceneTreeThe scene tree managing the game objects.
screenWidthThe width of the game screen.
screenHeightThe height of the game screen.

This function creates a new GhostMovementComponent and adds it to the specified ghost object. The component uses the provided Pac-Man object as a target for movement, and the scene tree for collision detection. The screen dimensions are used for boundary checking.
