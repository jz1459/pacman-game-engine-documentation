Detailed Description

Provides an abstraction layer for SDL2/SDL3 initialization and termination.

This module initializes the SDL library (and SDL_mixer) at the start of the application and ensures proper cleanup upon termination. It uses bindbc-sdl to load the SDL libraries dynamically, supporting multiple platforms (Windows, macOS, Linux).
